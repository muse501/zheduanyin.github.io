构建实例

构建实例是在 request.POST 的基础上，对其进行补充，从而获得一个完整的实例内容。

官方文档给出了这个例子：在 view 中使用 save(commit=False) 处理，然后补充或修改实例的相关 fields。如果实例已存在数据库的话，使用 update() 来更新数据库内容；如果是新建的条目，则使用 save() 来添加到数据库。

要完成更新一个不存在于数据库的操作需要

  1. 首先检测它是否已存在数据库
  2. 如果已存在，那么需要手动指定其 pk 来获得该实例
  3. 然后更改相关 field 内容
  4. 最后使用 update() 保存到数据库

Django 有一个 update_or_create 已经实现了上述所有功能，可以避免这个 try … except … 判断实例是否已存在
